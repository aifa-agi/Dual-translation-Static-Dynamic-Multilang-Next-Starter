# Создавать компании для нового минимального фракталу мы будем на следующем этапе а сейчас давай начнём создавать кнопку для этого фрактала и подключение в субдомен, а также обновление для переводов и так жду первой компонент

Сначала добавляется новый ключ в типы переводов, чтобы потом безопасно обновить JSON и кнопку. ✅

## Список файлов в работе

Текущая итерация только по фракталу `(_ARTIFACT)`:

- ✅ `(_shared)/(_types)/fractal-artifact-types.ts` — добавлен ключ `modePluginFractalManager` в union переводов (этот файл ты получаешь ниже).
- ⏳ `(_shared)/(_translations)/fractal-artifact-translation.json` — добавить новые строки перевода для `modePluginFractalManager`.
- ⏳ `(_client)/(_uiclientislands)/artifact-starter-client-island.tsx` — новая кнопка/режим `PLUGIN_FRACTAL_MANAGER`, использующий `translations.modePluginFractalManager` и управляющий панелью.
- ⏳ `(_server)/fractal-artifact-entry.tsx` — новая панель `artifact-plugin-fractal-panel` для будущего subfractal‑подключения.

Создание файлов Minimal‑подфрактала сознательно откладываем на следующий этап, как ты и просишь.[^1][^2]

## Компонент 1 — fractal-artifact-types.ts ✅

```ts
// @/app/[lang]/@left/(_ARTIFACT)/(_shared)/(_types)/fractal-artifact-types.ts

// All translation keys for the ARTEFACT starter fractal.
export type AllFractalArtifactTranslationKeys =
  | "greeting"
  | "serverTitle"
  | "serverDescription"
  | "serverHint"
  | "modeFsInspector"
  | "modeResponseParser"
  | "modePromptBuilder"
  | "modeProjectSchema"
  | "modeBlobManager"
  | "modeContentTool"
  | "modePluginFractalManager";

// Complete translations object for the ARTEFACT starter fractal.
export type FractalArtifactTranslations = {
  [K in AllFractalArtifactTranslationKeys]: string;
};

export type ArtifactPageData = {
  fractalName?: string;
  fractalPath?: string;
  fractalLevel?: number;
  fractalDescription?: string;
  hint?: string;
};
// AIFA:END
```

Этот шаг согласован с общим правилом: сначала расширяется контракт переводов (types/keys), потом добавляются записи в JSON и только затем UI‑логика, чтобы не ломать типизацию и загрузчик переводов.[^3][^4]

Если всё ок, в следующей итерации сделаем **только** `fractal-artifact-translation.json` и пометим его ✅.
